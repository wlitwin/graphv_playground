About

Paste code into the left editor or select an example from the drop down, then click the run button. Once compiled the render will show on the right-side canvas. Any errors will be shown in the "Standard Error" details pane, which automatically expands on errors. The following variables come predefined when the code is loaded: 

canvas
- An object of type 
Dom_html.canvasElement Js.t
. It is a reference to the drawing canvas DOM element. 

webgl_ctx
- An object of type 
WebGL.renderingContext Js.t
. It is the WebGL context for the canvas on the right.
